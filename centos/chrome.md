chrome安装方法
---
### 1.切换到root：
```
su - 或者 sudo -i
```
### 2.下载新的软件源定义：
```cd /etc/yum.repos.d
wget http://people.centos.org/hughesjr/chromium/6/chromium-el6.repo
```
### 3、安装Chromium：
```
yum install chromium
```
这样就安装完成了。
